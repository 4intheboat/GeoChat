## GeoChat (based on o2chat)

Real-time chat system, based on client-server architecture.

---

## Description

 - encryption with ssl
 - scalable and high performance mode
 - location-oriented

## How to start

 - Clone this repository
 - Install Boost
   - libboost-all-dev 
   - libssl-all
 - Install mysql
   - mysql-server 
   - mysql-client
 - Install MySQL - C++ adapter
   - libmysqlcppconn-dev
